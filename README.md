# 🎓 Student Stress Level Prediction

A Machine Learning web application built using **Python**, **Scikit-learn**, and **Streamlit** to predict whether a student has **High Stress** or **Low Stress** based on lifestyle and academic factors.

## 🚀 Live Demo
https://student-stress-level-prediction-4g2imsayluyacurjqzd6t8.streamlit.app

## 📂 GitHub Repository
https://github.com/satyam8516/Student-Stress-Level-Prediction

## ✨ Features
- Student Type Selection
- Sleep Hours Input
- Study Hours Input
- Social Media Hours
- Attendance Percentage
- Exam Pressure
- Family Support
- Predict High/Low Stress

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Joblib

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
